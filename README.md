# Happy Birthday Bubie

Website ulang tahun romantis berbentuk surat digital.

## Cara menjalankan

Buka file `index.html` langsung di browser.

## Ganti foto

Semua placeholder ada di folder `assets` dalam format JPG. Kamu bisa mengganti file JPG placeholder dengan foto sendiri, atau ubah `src` pada `index.html` di bagian yang diberi komentar `ganti foto di sini`.

## Tambah video ucapan

Taruh video ucapan di folder `assets` dengan nama `video-ucapan.mp4`.

Kalau nama file videonya berbeda, ubah bagian ini di `index.html`:

```html
<source src="assets/video-ucapan.mp4" type="video/mp4" />
```

## Tambah lagu

Taruh lagu di folder `assets` dengan nama `lagu.mp3`.

Kalau nama file lagunya berbeda, ubah bagian ini di `index.html`:

```html
<source src="assets/lagu.mp3" type="audio/mpeg" />
```
